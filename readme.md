# 덜 편한 가계부
## 소개  
현재 잘 사용중인 편한 가계부 라는 앱을 클론코딩해서  
Room이나 Koin과 같은 기술스택들을 공부해볼겸 시작해본 프로젝트입니다.

* Third Party  
  - Kotlin Coroutine
  - Koin
